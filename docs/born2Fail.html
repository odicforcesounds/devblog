In the perfect world, Everyone would accept and would enjoy contribute to the cause, accepting all the restrictions that this application ask to everyone.

How useful it is? Not useful at all. How usable it is? Not at all. The application is a joke (and if is not, you should see it as a joke). The application don't solve any problem, in fact just add new problems.

This is the problem of dreams. They ignore reality.
