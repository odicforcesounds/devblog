A 14GB ISO image

The Security Onion Solutions Linux distribution looks awesome but when I read the Download section on their GitHub, they care to mark ISO Validation and Verification quality, saying that, if the ISO is not verified, we can download from other mirror, while I can see only one, I will not download the ISO many times. 14GB is a lot, but the Web interface they create with all those tools are really great.

This distro is announcing some dedicated hardware to use SOS Linux distro. Since the main objective is to fullfil the hard drive with Network data, I will not use it, but at least I cannot deny that they try to create something new, within Linux usage. There are a few more distros that are similar to this one, but not so complete. Managing VMs, is something that some other WUI's do but have less menu/colors quality that others.

I really like the Minimal perspective on Linux distros and for my case I will not use this one, but I may approve it, since they try to offer solutions that can be useful for some Professional cases.
