Mainly the Password Managers. I use KeepassXC and I HATE the option to enable to share anything Online, while in Gentoo, we can unable this stupid option, that doesn't make any sense to exist. I don't know what is the intentions of any of the above sites application and I cannot guaratee that they are total secure. So, to someone that have some privacy / security problems, using a Online Password Manager is like shuting the foot.

I try to save and backup everything I need to a hard drive, that is 99% of the time, offline. I deny to save passwords in the Browser and in any other way that can be exposed online. The best I used to use, was encrypted files using GPG and expose them in Google Driver for example, but since I don't know if Google is trustful, I remove them from there. No is Not!!! Sometimes I think this blog is based on Window$ users, since is always talking about the ESET anti-virus that is another Window$ crap. I love Windows, Offline. Is the best OS for gamming, Offline. The best OS (offline). Always offline is secure and fast. No AV is needed and no obvious danger can happen on this machine.
