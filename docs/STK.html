STK is a Verification System of Others Actions and Qualities

Using STK, you are able to register your Google account into this application, upload videos on Youtube within this account and validate your actions, that focus clear objectives: Animal care, nature care and human care. All users can create simple videos of simple actions, and they can create Challenges they cannot do it alone. All users can do from 0 to 255 or more actions, for each objective. 3 objectives 3 RGB channels. Those are the colors of your profile.

An extra feature is added to all users. Each user have a QR code that identify the user by other user that use the same application. As you can imagine, users need to accept to share their QR code to be validated by others in many different contexts like civism, family, work and others. Each context can have 10 or more qualities that can be selected to add +1 or -1 value to that "personal quality".

Group registration is possible to solve the Challenges that individuals alone, cannot. By solving challenges, groups are announced the main website of this application, with their name, website, number of users, solved challenge videos.

By this conditions, we create a World Wide Challenge for Image Propose only, or in another words, a Logo System competition of self promotion. With this system, users can have a better idea of how others see them, in different moments of time, within the different contexts of society.
