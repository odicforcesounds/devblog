Being Clear and Objective, feels Good

I am trying to develop a new application, inspired by another application, with something different from all others that offer the same "Service". Service is in Quotes, because my application will be like a Game, but at the same time, if reach popularity can offer some kind of service, even if is not a service but user activity content that is useful for services...

Anyway, this team suggest me to use Jira Platform. After one week I as feeling that I was the only person using that platform and their ability to streucture the project steps were really bad, since they were creating a Main Task, that have many sub-tasks and for each sub-task, many other sub-tasks, without any description, without a single note or link to help developers decisions and coding. Just "objectives" like "add option A", "add option B". Everything very vague...

Since I start using Jira, I start to learn by intuition but a little bit more, after listen some YouTube video that explain some details. How they work and how useful they can be for the developers and even for me as a client, not only to monitor their activity, but to understand how the application logic was being built... facing the idea that had a plan... but they don't like plans too.

So, I made some pressure (Big Pressure)... and after some Days, they start doing something better.. more descriptive Tasks... With the correct Tag, using the Correct State.. and with that attitude I feel better... My frustration Level on today was really High... I was feeling that, Or I was not able to explain them their dutty using a platform like Jira or they were just ignoring me...
