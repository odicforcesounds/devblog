Because using Links to install Gentoo we cannot see Warnings or Alerts

Last Gentoo Linux Review I describe how confusing can happen when installing Gentoo from the Minimal ISO. Then I face some issues that talk in the same page what works for Systemd and what works for OpenRC, while Warnings are not shown as warnings and therefor some details are not clear enought for me.

Then, I start reading the Full Installation Alpha Version of their documentation and I realize that warnings were there, while the Systemd information too. So at this point, I don't wish to follow Gentoo Handbook, but build my own by taking notes of their Handbook notes. Is the difference between using Firefox or w3m/links/elinks/lynx Terminal Web-Browsers.

For example, I was complainming about Memory Usage at compilation Time, but they have a Warning saying that using MAKEOPTS="-j2" is the safest way to not use all memory that we have in our machine, which can run out of memory if we use a bigger number while building something like Rust or GCC. Our Machine will crash and forcing shutdown is necessary. Is a Installation Skill that Kills Installation. :D

Another curiosity is the Sakaki (some Japanese user that share his installation process) with a lot of improvements if compared to the Default and Official Gentoo Manual, while the first warning tell us to Not Follow that Guide. Then I ask myself: why that Guide is still there anyway? I don't know, but if I assume that some value is out of date, I would hide it, to not confuse users that use "Links" to read the manual pages available on Gentoo websites.
