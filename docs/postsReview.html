Checking blog activity and thinking about History

I always like to share some things that I use. This blog, somewhere is Time, was just a Markdown site, to share some things about Technology that are common for many. There is nothing special on this files, but can be, since depends on you, not on me.

Calcurse is a simples agenda + todo oraganizer. Toot is a Mastodon client. IRSSI that is a IRC client, very useful tool and, let us be thankful to all communities that maintain them. Tmux that is used to manage all these applications running in one single terminal window. Vim that is an text editor with simplicity in mind, W3M to browser the static web, full of text to read and without any kind of animations or publicity (no ads), Git in case you use some public code hosting to maintain your blogs or services, like this blog is, MPV to listen some music or enjoy some movie to inspire your activity, and a list of Awesome Self Services that anyone can run on their computers. In actual days, you cannot use Gmail in a secure way, using Mutt or maybe you can, see the change logs, but is useful to manage several emails accounts. Reading the news without opening a web browser is what I like most, therefor Newsboat is the Yellow one.

In a old 32bits computer everything works just fine. Could be the New generation ones, and with one single terminal window, I can use everything I need to use a computer
