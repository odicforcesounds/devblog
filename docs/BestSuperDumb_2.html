FreeBSD Basics: pkg

Binary Packages is what pkg can install.

Run pkg in the terminal and install or update pkg package.

pkg help install
pkg update -f
pkg upgrade
pkg search string
pkg install string
pkg info
pkg audit -F
pkg delete string
pkg autoremove
pkg prime-list
pkg prime-origins
pkg clean
pkg lock string
pkg unlock string
