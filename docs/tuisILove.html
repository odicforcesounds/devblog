Wiki-Tui & Tui-Journal

Both application use Rust Lang. Be sure you install Rust

The Wiki-Tui application written in Rust is really cool, because I can explore articles from wikipedia, without leaving the terminal. You can change the colors of links and other details, to be more readable. Using a normal web browser like the w3m, Lynx, Elinks or Links, text will not be so well formated, while with this app, everything looks pretty cool.

#!/bin/bash
# Install Wiki-Tui using Rust
cargo install --git https://github.com/Builditluc/wiki-tui wiki-tui --locked

The Tui-Journal application is to organize your notes in different "pages", associated with Tags. I see it useful, even if I can use Vim to organize my notes without depending on a external application. Even then, I like this application too, that is writen using Rust.

#!/bin/bash
# Install Tui-Journal
cargo install --git https://github.com/AmmarAbouZor/tui-journal --locked
