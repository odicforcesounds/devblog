Wiki-Tui & Tui-Journal

Both application use Rust. Be sure you install it first

Before trying to install, install the necessay packages too:
apt install cmake binutils make pkg-config libssl-dev -y

#!/bin/bash
# Install Rust as a user without admin priviledges
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

The Wiki-Tui application written in Rust is really cool, because I can explore articles from Wikipedia.org, without leaving the terminal. You can change the colors of links and other details, to be more readable. Using a normal web browser like the w3m, Lynx, Elinks or Links, text will not be so well formated, while with this app, everything looks pretty cool.

#!/bin/bash
# Install Wiki-Tui using Rust
cargo install --git https://github.com/Builditluc/wiki-tui wiki-tui --locked

The Tui-Journal application is to organize your notes in different "pages", associated with Tags. I see it useful, even if I can use Vim to organize my notes without depending on a external application. Try it. .

#!/bin/bash
# Install Tui-Journal
cargo install --git https://github.com/AmmarAbouZor/tui-journal --locked
