Learning to use Vim

The best option is always to install Vim, open and type :help. You will start with the basis, while you will see that documentation is big too. Using Vim can be speed up our work. When opening a new file with Vim, type i to start writting and use ESC keyboard to go back to visual mode. When you type i you are using Vim in Insert Mode. Type :w to save changes and :q to close file or leave editor.

For example, today I figure out that I wish to add a HTML class HTML class="textbox". I just type vim * into the docs/ directory and all posts where there, in one single page. Then you ask: how can the editor open more than 300 files and we only see one? And the answer is buffers.

Some of the default values that most vi users like to setup on the .vimrc is the set number. To record a macro you need to press q in visual mode, to start and create the sequence you need to do in many files, and press q to stop recording. Use the . to repeat the last changes into all other files. Select buffers by using the command buffer number-of-buffer. For example :buffer 13 or the shortcut :b 13 will open the 13 file in buffer. Use :buffers to list them all. You can use :bnext or bpreviews to jump between open files.

If you have a Screen that have big resolution, you can easly work in several files using the vsplit option to open new files in a vertical direction and split in a horizontal one. You can use tabs too. Use / in visual mode to find some text and use :%s/text-to-modify/modified-text/g to change all text that is needed to change. Since in this real example, I only need to find the HTML class, I created a Macro to apply the same commands (select and add new content) within all files. In few minutes, I already change everything. Just open Vim and type :. Now using the TAB explore all available commands.
