The bad side of some Nixovena Distros

While Zen Clora Linux works as expected, the Overkill Linux present problems connecting into the internet using in Live Mode, the HardenWing does not connect to the internet when trying to install. No live mode on this last distro, which means that the Harden distros that I test so far, present some problems that many others don't.

Zenclora Linux use transparency in all menus and only takes 700MB of RAM to use it. Pretty normal consume if I compare it to a normal Debian installation with MATE Desktop. Therefor, the ZenClora Linux is approved, while Overkill and HardenWing Linux are not.
