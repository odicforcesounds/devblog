Nixovena is the group name

Sometimes I like to see what Linux/BSD distros Distrowatch update and few posts before, I just said that not all Linux distros add something new to them, but personalize what they can, giving new colors to the GUI. Some of them are dedicated to Offensive Security and others to Defensive Security which is what Security is about at the end of all. Defensive creates Security.

Anyway, Nixovena is a group of people that made different Linux distros with different objectives, which is a sign of Creativity. So, I will test their Almighty distro that looks pretty good for privacy/security in the web, the LengToolKit which looks like a Rescue/Repair Linux distro, which I always like to have in case of having to deal with those kind of problems, the HardenWing that looks like a Defensive Security dedicated approach and the Fishy distro that is optimized for old/legacy machines. I don't know if I will test the OverKill version, since I don't like to Kill anything at all, while sometimes is necessary, when they are only... Software!
